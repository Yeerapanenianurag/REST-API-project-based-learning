# Django REST API Project

A RESTful API service built with Django and Django REST Framework. This app demonstrates basic CRUD operations, authentication, and common API patterns.

## Table of Contents
- Features
- Technologies Used
- Installation
- Usage
- API Endpoints
- License

## Features
- CRUD for main resources
- Token-based authentication
- Filtering, pagination, and search
- Admin interface for management

## Technologies Used
- Python 3.10+
- Django 4.x
- Django RE
- REST Framework
- Postgres SQL(default, easily switchable to PostgreSQL)
